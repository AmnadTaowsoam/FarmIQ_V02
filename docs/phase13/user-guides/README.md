## Getting Started

### First Login

1. **Access the Platform**
   - Navigate to [https://farmiq.example.com](https://farmiq.example.com)
   - Enter your email and password
   - Click "Sign In"

2. **Complete Your Profile**
   - Fill in your name and contact information
   - Set your preferred language (English/Thai)
   - Configure timezone settings

3. **Dashboard Tour**
   - Welcome to your FarmIQ dashboard
   - Overview of key metrics and quick actions

### Setting up Your Profile

- Navigate to **Settings** > **Profile**
- Update your personal information
- Configure notification preferences
- Set up two-factor authentication (recommended)

### Navigating the Dashboard

| Section | Description |
|---------|-------------|
| **Overview** | Key KPIs at a glance |
| **Farms** | Manage your farms and barns |
| **Feeding** | Feed intake and nutrition tracking |
| **WeighVision** | AI-powered weight monitoring |
| **Reports** | Analytics and insights |
| **Settings** | Account and system configuration |

## Farm Management

### Creating a New Farm

1. Click **Farms** in the navigation menu
2. Click **+ New Farm** button
3. Fill in the farm details:
   - Farm Name
   - Location (address, coordinates)
   - Farm Type (broiler, layer, swine, etc.)
   - Contact Information
4. Click **Create Farm**

### Managing Barns

#### Adding a Barn

1. Select your farm from the Farms list
2. Click **+ Add Barn**
3. Enter barn information:
   - Barn Name/Number
   - Capacity (number of animals)
   - Barn Type
   - Dimensions
4. Configure sensor assignments
5. Click **Save**

#### Barn Configuration

- **Temperature Sensors**: Set alert thresholds
- **Humidity Sensors**: Configure optimal ranges
- **Feed Systems**: Link to feed management
- **Water Systems**: Monitor water consumption

### Device Setup

#### Connecting IoT Sensors

1. Navigate to **Settings** > **Devices**
2. Click **+ Add Device**
3. Scan QR code or enter device ID
4. Assign device to a specific barn
5. Verify connection status

#### WeighVision Camera Setup

1. Position camera in designated area
2. Connect to power and network
3. Register camera in FarmIQ
4. Calibrate using reference object
5. Test weight detection

## Daily Operations

### Recording Feed Intake

#### Manual Feed Recording

1. Go to **Feeding** > **Feed Intake**
2. Select the barn
3. Enter feed amount (kg)
4. Select feed type
5. Add notes if needed
6. Click **Record**

#### Automated Feed Recording

- Feed systems automatically record consumption
- Data syncs in real-time
- View feed efficiency metrics

### WeighVision Sessions

#### Starting a WeighVision Session

1. Navigate to **WeighVision** > **Sessions**
2. Click **+ New Session**
3. Select barn and camera
4. Set session duration
5. Click **Start Session**

#### Monitoring Live

- Real-time weight detection
- Animal counting
- Growth tracking
- Alert notifications

### Health Records

#### Recording Health Events

1. Go to **Health** > **Records**
2. Click **+ Add Record**
3. Select animal or group
4. Choose event type:
   - Vaccination
   - Medication
   - Mortality
   - Disease outbreak
5. Enter details and save

## Reports & Insights

### Generating Reports

1. Navigate to **Reports**
2. Select report type:
   - Daily Summary
   - Weekly Analysis
   - Monthly Performance
   - Custom Report
3. Set date range
4. Choose filters (farm, barn, metrics)
5. Click **Generate**

### Understanding KPIs

#### Key Performance Indicators

| KPI | Description | Target Range |
|-----|-------------|--------------|
| **FCR** | Feed Conversion Ratio | 1.5 - 1.8 |
| **ADG** | Average Daily Gain | 40-60 g/day |
| **Mortality Rate** | Death percentage | < 2% |
| **Weight Uniformity** | Weight distribution CV | < 10% |

### AI Recommendations

FarmIQ AI provides actionable insights:

- **Feed Optimization**: Adjust feed composition
- **Health Alerts**: Early disease detection
- **Environmental**: Temperature/humidity adjustments
- **Growth Predictions**: Expected harvest dates

## Troubleshooting FAQ

### Common Issues

#### Q: I can't log in to my account

**A**: Try the following:
1. Check your email and password
2. Clear browser cache and cookies
3. Reset password if needed
4. Contact support if issue persists

#### Q: Sensor data is not updating

**A**: Verify:
1. Device has power
2. Network connection is stable
3. Device is registered correctly
4. Check device status in Settings > Devices

#### Q: WeighVision not detecting animals

**A**: Check:
1. Camera positioning and lighting
2. Calibration is correct
3. No obstructions in view
4. Camera firmware is up to date

#### Q: Reports are not generating

**A**: Ensure:
1. Date range is valid
2. Data exists for selected period
3. Filters are correct
4. Try clearing cache and retry

## Appendix

### Glossary

- **FCR**: Feed Conversion Ratio - feed consumed per unit of weight gain
- **ADG**: Average Daily Gain - weight increase per day
- **IoT**: Internet of Things - connected sensors and devices
- **WeighVision**: AI-powered visual weight estimation system
