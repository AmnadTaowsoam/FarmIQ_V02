## เริ่มต้นใช้งาน

### การเข้าสู่ระบบครั้งแรก

1. **เข้าถึงแพลตฟอร์ม**
   - ไปที่ [https://farmiq.example.com](https://farmiq.example.com)
   - ป้อนอีเมลและรหัสผ่าน
   - คลิก "เข้าสู่ระบบ"

2. **กรอกข้อมูลโปรไฟล์**
   - กรอกชื่อและข้อมูลติดต่อ
   - ตั้งค่าภาษาที่ต้องการ (อังกฤษ/ไทย)
   - กำหนดการตั้งค่าเขตเวลา

3. **แนะนำแดชบอร์ด**
   - ยินดีต้อนรับสู่แดชบอร์ด FarmIQ ของคุณ
   - ภาพรวมของตัวชี้วัดสำคัญและการกระทำด่วน

### การตั้งค่าโปรไฟล์

- ไปที่ **การตั้งค่า** > **โปรไฟล์**
- อัปเดตข้อมูลส่วนบุคคล
- กำหนดค่าการแจ้งเตือน
- ตั้งค่าการรับรองตัวตนสองปัจจัย (แนะนำ)

### การนำทางแดชบอร์ด

| ส่วน | คำอธิบาย |
|------|-------------|
| **ภาพรวม** | ตัวชี้วัดสำคัญในภาพรวม |
| **ฟาร์ม** | จัดการฟาร์มและโรงเลี้ยงของคุณ |
| **การให้อาหาร** | ติดตามการบริโภคอาหารและโภชนาการ |
| **WeighVision** | การตรวจสอบน้ำหนักด้วย AI |
| **รายงาน** | การวิเคราะห์และข้อมูลเชิงลึก |
| **การตั้งค่า** | การกำหนดค่าบัญชีและระบบ |

## การจัดการฟาร์ม

### การสร้างฟาร์มใหม่

1. คลิก **ฟาร์ม** ในเมนูนำทาง
2. คลิกปุ่ม **+ ฟาร์มใหม่**
3. กรอกรายละเอียดฟาร์ม:
   - ชื่อฟาร์ม
   - ตำแหน่งที่ตั้ง (ที่อยู่ พิกัด)
   - ประเภทฟาร์ม (ไก่เนื้อ ไก่ไข่ สุกร ฯลฯ)
   - ข้อมูลติดต่อ
4. คลิก **สร้างฟาร์ม**

### การจัดการโรงเลี้ยง

#### การเพิ่มโรงเลี้ยง

1. เลือกฟาร์มของคุณจากรายการฟาร์ม
2. คลิก **+ เพิ่มโรงเลี้ยง**
3. ป้อนข้อมูลโรงเลี้ยง:
   - ชื่อ/หมายเลขโรงเลี้ยง
   - ความจุ (จำนวนสัตว์)
   - ประเภทโรงเลี้ยง
   - ขนาด
4. กำหนดค่าการกำหนดเซ็นเซอร์
5. คลิก **บันทึก**

#### การกำหนดค่าโรงเลี้ยง

- **เซ็นเซอร์อุณหภูมิ**: ตั้งค่าเกณฑ์การแจ้งเตือน
- **เซ็นเซอร์ความชื้น**: กำหนดช่วงที่เหมาะสม
- **ระบบอาหาร**: เชื่อมโยงกับการจัดการอาหาร
- **ระบบน้ำ**: ติดตามการบริโภคน้ำ

### การตั้งค่าอุปกรณ์

#### การเชื่อมต่อเซ็นเซอร์ IoT

1. ไปที่ **การตั้งค่า** > **อุปกรณ์**
2. คลิก **+ เพิ่มอุปกรณ์**
3. สแกน QR Code หรือป้อนรหัสอุปกรณ์
4. กำหนดอุปกรณ์ให้กับโรงเลี้ยงที่ระบุ
5. ตรวจสอบสถานะการเชื่อมต่อ

#### การตั้งค่ากล้อง WeighVision

1. วางกล้องในพื้นที่ที่กำหนด
2. เชื่อมต่อกับไฟและเครือข่าย
3. ลงทะเบียนกล้องใน FarmIQ
4.สอบเทียบโดยใช้วัตถุอ้างอิง
5. ทดสอบการตรวจจับน้ำหนัก

## การดำเนินงานประจำวัน

### การบันทึกการบริโภคอาหาร

#### การบันทึกอาหารด้วยตนเอง

1. ไปที่ **การให้อาหาร** > **การบริโภคอาหาร**
2. เลือกโรงเลี้ยง
3. ป้อนปริมาณอาหาร (กก.)
4. เลือกประเภทอาหาร
5. เพิ่มหมายเหตุหากจำเป็น
6. คลิก **บันทึก**

#### การบันทึกอาหารอัตโนมัติ

- ระบบอาหารบันทึกการบริโภคโดยอัตโนมัติ
- ข้อมูลซิงค์แบบเรียลไทม์
- ดูตัวชี้วัดประสิทธิภาพอาหาร

### เซสชัน WeighVision

#### การเริ่มเซสชัน WeighVision

1. ไปที่ **WeighVision** > **เซสชัน**
2. คลิก **+ เซสชันใหม่**
3. เลือกโรงเลี้ยงและกล้อง
4. ตั้งค่าระยะเวลาเซสชัน
5. คลิก **เริ่มเซสชัน**

#### การตรวจสอบแบบสด

- การตรวจจับน้ำหนักแบบเรียลไทม์
- การนับสัตว์
- การติดตามการเจริญเติบโต
- การแจ้งเตือน

### บันทึกสุขภาพ

#### การบันทึกเหตุการณ์สุขภาพ

1. ไปที่ **สุขภาพ** > **บันทึก**
2. คลิก **+ เพิ่มบันทึก**
3. เลือกสัตว์หรือกลุ่ม
4. เลือกประเภทเหตุการณ์:
   - การฉีดวัคซีน
   - การให้ยา
   - การตาย
   - การระบาดของโรค
5. ป้อนรายละเอียดและบันทึก

## รายงานและข้อมูลเชิงลึก

### การสร้างรายงาน

1. ไปที่ **รายงาน**
2. เลือกประเภทรายงาน:
   - สรุปรายวัน
   - การวิเคราะห์รายสัปดาห์
   - ประสิทธิภาพรายเดือน
   - รายงานแบบกำหนดเอง
3. ตั้งช่วงวันที่
4. เลือกตัวกรอง (ฟาร์ม โรงเลี้ยง ตัวชี้วัด)
5. คลิก **สร้าง**

### การทำความเข้าใจ KPI

#### ตัวชี้วัดประสิทธิภาพหลัก

| KPI | คำอธิบาย | ช่วงเป้าหมาย |
|-----|-------------|--------------|
| **FCR** | อัตราแปลงอาหาร | 1.5 - 1.8 |
| **ADG** | อัตราการเจริญเติบโตเฉลี่ยต่อวัน | 40-60 กรัม/วัน |
| **อัตราการตาย** | เปอร์เซ็นต์การตาย | < 2% |
| **ความสม่ำเสมอของน้ำหนัก** | การกระจายน้ำหนัก CV | < 10% |

### คำแนะนำจาก AI

FarmIQ AI ให้ข้อมูลเชิงลึกที่สามารถดำเนินการได้:

- **การเพิ่มประสิทธิภาพอาหาร**: ปรับองค์ประกอบอาหาร
- **การแจ้งเตือนสุขภาพ**: การตรวจจับโรคในระยะแรก
- **สิ่งแวดล้อม**: การปรับอุณหภูมิ/ความชื้น
- **การคาดการณ์การเจริญเติบโต**: วันที่เก็บเกี่ยวที่คาดไว้

## การแก้ปัญหา

### ปัญหาทั่วไป

#### ถาม: ฉันไม่สามารถเข้าสู่ระบบบัญชีของฉันได้

**ตอบ**: ลองดังนี้:
1. ตรวจสอบอีเมลและรหัสผ่านของคุณ
2. ล้างแคชและคุกกี้ของเบราว์เซอร์
3. รีเซ็ตรหัสผ่านหากจำเป็น
4. ติดต่อฝ่ายสนับสนุนหากปัญหายังคงอยู่

#### ถาม: ข้อมูลเซ็นเซอร์ไม่อัปเดต

**ตอบ**: ตรวจสอบ:
1. อุปกรณ์มีไฟ
2. การเชื่อมต่อเครือข่ายเสถียร
3. อุปกรณ์ลงทะเบียนถูกต้อง
4. ตรวจสอบสถานะอุปกรณ์ในการตั้งค่า > อุปกรณ์

#### ถาม: WeighVision ไม่ตรวจจับสัตว์

**ตอบ**: ตรวจสอบ:
1. ตำแหน่งกล้องและแสงสว่าง
2. การสอบเทียบถูกต้อง
3. ไม่มีสิ่งกีดขวางในมุมมอง
4. เฟิร์มแวร์กล้องเป็นปัจจุบัน

#### ถาม: รายงานไม่ถูกสร้าง

**ตอบ**: ตรวจสอบให้แน่ใจ:
1. ช่วงวันที่ถูกต้อง
2. มีข้อมูลสำหรับช่วงเวลาที่เลือก
3. ตัวกรองถูกต้อง
4. ลองล้างแคชและลองอีกครั้ง
